Working with a library

The API has been set up to be as accessible as possible, but working with a library is often easier. On this page, you can find the currently available libraries.

Would you like to use a library in a different language? We could potentially set this up for you. Please email us at api@laposta.nl.

PHP

The code for our easy-to-use PHP wrapper is available on Github.

The library contains a folder with examples. Please also refer to the documentation for examples in php.

.NET (C#)

The code for our easy-to-use .NET wrapper is available on Github.

This is a Visual Studio solution, compatible with Visual Studio versions 2008 up to and including 2012. The project contains a folder with examples. Please also refer to the documentation for examples in .NET.
